Helix — Visual Identity Exploration v1

Performance-focused, coach-respecting, intelligent but not clinical. Positioned between premium sport tech and high-performance medical.

Brand Territory

Helix is built for elite coaching businesses that demand precision, scalability, and intelligence. The visual identity should reflect three core values: authority through data, respect for human expertise, and premium engineering. Every pixel serves the coach and client experience—nothing feels clinical or corporate. The design sits confidently between high-performance sport tech and intelligent medical-grade systems.

Typography Direction

Three typefaces working in harmony to create a clear, premium visual hierarchy.

Display Font
Inter Bold
Used for headings, titles, and prominent labels. Geometric sans-serif for a modern, authoritative presence.
Body Font
The quick brown fox jumps over the lazy dog. Inter at 15px/24px baseline is the standard for readable body copy, UI labels, and interface text.
Perfect balance of legibility and premium feel across mobile and desktop.
Monospace Font
var metrics = { rpe: 8, volume: 245 };
JetBrains Mono for data points, code snippets, and technical metrics. Ensures clarity in performance numbers.
